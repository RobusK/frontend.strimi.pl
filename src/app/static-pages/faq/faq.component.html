{{ 'Podstawy' }}

{{ 'Co oznacza słowo "Strimi"?' }}

{{ 'Strimi to spolszczenie angielskiego słowa "stream" (strumień). Strimi to tematyczna rzeka różnych informacji.' }}

{{ 'Czym jest Strimi?' }}

{{ 'Strimi to miejsce do publikowania treści w tematycznych tagach, które są kreowane przez naszą Społeczność. Serwis jest oparty o zdecentralizowany blochkchain Steem. Strimi to platforma podobna do popularnego serwisu Reddit. Różnica polega na tym, że wszystkie treści na Strimi przechowywane są w sieci Steem (co raz zostanie dodane do sieci Steem, na zawsze tam pozostaje – nie da się nic usunąć). Blockchain Steem umożliwia stworzenie portalu społecznościowego w zdecentralizowanej sieci peer-to-peer.' }}

{{ 'Czym jest blockchain Steem?' }}

{{ 'Steem został stworzony w 2016 roku przez Neda Scotta i Dana Larimera (twórcę giełdy Bitshares). Ideą Steem jest stworzenie platformy social media opartej o technologię blockchain. Umożliwia to nagradzanie Użytkowników sieci za aktywność w postaci postów i komentarzy w platformie społecznościowej. Nagroda przesyłana jest za pomocą tokenów o nazwie Steem. Sieć Steem oprócz tokenu Steem posiada również drugą walutę SteemDollars (SBD), której kurs w odróżnieniu od Steem jest ściśle powiązany z kursem USD.' }}

{{ 'Czym blockchain Steem różni się od Bitcoina?' }}

{{ 'Główna różnica między tymi dwoma kryptowalutami jest taka, że celem Bitcoina jest wydobycie (kopanie) i transfer waluty, zaś Steem wydobywa się za pomocą metody Proof of Stake. Steem posiada dwa tokeny, Steem oraz SteemDollars i za ich pomocą nagradzani są nasi Użytkownicy.' }}

{{ 'Czym jest metoda Proof of Stake (PoS)?' }}

{{ 'PoS jest to forma miningu, która opiera się przede wszystkim na wynagradzaniu Użytkowników za sam fakt posiadania przez nich waluty w portfelu.' }}

{{ 'Treści i Nagrody' }}

{{ 'Kto dodaje treści?' }}

{{ 'Strimi to serwis społecznościowy, treści tworzone są przez naszych Użytkowników.' }}

{{ 'Jak dodać treść?' }}

{{ 'Po zalogowaniu na górnej belce serwisu znajduje się przycisk „Dodaj treść” klikamy w niego i następnie mamy dwie możliwości dodania treści, treść własną lub jako link do strony, grafiki lub materiału wideo. Warto tutaj zaznaczyć aby podczas dodawania nowej treści dodać tytuł zgodny ze stanem faktycznym. W opisie treści w kilku zdaniach opisać docelowy link, im ciekawszy opis tym więcej Użytkowników przyciągnie.' }}

{{ 'Jak wybierać nagrody podczas dodawać treści?' }}

{{ 'Jak już wiemy 75% z nagrody trafia do autora, nagrody otrzymujemy podczas dodawania treści, jednak mamy wybór jaką nagrodę chcemy otrzymać po 7 dniach, mamy do wyboru trzy możliwości:' }}

  • {{ '50%/50%' }}{{ ' - 50% nagrody otrzymamy w SteemPower, zaś drugie 50% w SteemDollars,' }}
  • {{ '100% SteemPower ' }}{{ ' – 100% nagrody trafia do nas w postaci SteemPower,' }}
  • {{ 'Rezygnacja' }}{{ ' – całkowicie zrzekamy się nagrody.' }}

{{ 'Zalecana obecnie nagroda jest 50%/50% z przyczyn ekonomicznych.' }}

{{ 'Czym są liczby obok treści?' }}

{{ 'Są to sumy głosów oddanych na daną treść przez naszych Użytkowników.' }}

{{ 'Czym są liczby z przedrostkiem $ obok treści?' }}

{{ 'Są to sumy tokenów, które otrzyma autor oraz głosujący (tzw. kuratorzy) na daną treść po 7 dniach od daty publikacji treści (treść przez 7 dni „zbiera głosy”). Nagrody zdobywane są poprzez pisanie treści lub komentarzy. Po siedmiu dniach (od daty publikacji) dana treść zostaje "ukryta", aby inne treści mogły wskoczyć na jej miejsce.' }}

{{ 'Skąd się biorą nagrody?' }}

{{ 'Blockchain Steem (z własnej puli) rozdziela tokeny proporcjonalnie do głosów, jakie zostały oddane przez Użytkowników na wszystkie treści, jednak mając na uwadze siłę/moc głosu Użytkowników (SteemPower). Każda nagroda z danej treści dzieli się na dwie części, 75% z nagrody trafia do autora, zaś pozostałe 25% do głosujących na tą treść.' }}

{{ 'Jak głosować?' }}

{{ '25% z nagrody danej treść jest rozdane do Wszystkich kuratorów (głosujących). Istnieją tutaj pewne możliwości jak osiągnąć maksymalne korzyści z głosowania na treści. Każda treść po dodaniu ma tzw. 30\'minutowe okienko czasowe, głosowanie wygląda następująco (0 minut = czas publikacji treści):' }}

  • {{ '0 min.' }}{{ ' - 100% Twojej nagrody otrzymuje autor treści,' }}
  • {{ '3 min.' }}{{ ' - 90% Twojej nagrody trafia do autora, 10% do Ciebie,' }}
  • {{ '15 min.' }}{{ ' - 50% Twojej nagrody trafia do autora, 50% do Ciebie,' }}
  • {{ '27 min.' }}{{ ' - 10% Twojej nagrody trafia do autora, 90% do Ciebie,' }}
  • {{ '30 min. i więcej' }}{{ ' - 100% Twojej nagrody trafia do Ciebie.' }}

{{ 'Statystyki pokazują, że najlepszy czas na głosowanie jest między 25 a 30 minutą od czasu publikacji treści, jednak nie jest to regułą.' }}

{{ 'Jak odbierać nagrody?' }}

{{ 'Twoje nagrody możesz odbierać w zakładce „portfel” w swoim profilu po 7 dniach od daty publikacji treści. Strimi pobiera piętnaście procent z nagrody wyrażonej w SteemPower.' }}

{{ 'Limity' }}

{{ 'Blockchain Steem narzuca odgórnie limity w postaci:' }}

  • {{ '5 minut' }}{{ ' - pomiędzy dodawaniem treści,' }}
  • {{ '20 sekund' }}{{ ' - pomiędzy dodawaniem komentarzy,' }}
  • {{ '3 sekundy' }}{{ ' - pomiędzy głosowaniem na treści i komentarze.' }}

{{ 'Pozostałe' }}

{{ 'Czym jest SteemPower (SP)?' }}

{{ 'SteemPower jest to siła głosu, dla przykładu posiadając 380SP i głosując 100% na daną treść, Twój głos będzie warty około $0.10. SteemPower możemy zdobywać poprzez dodawanie i głosowanie na treści czy komentarze. Aktualna cena rynkowa Steem również mocno wpyłwa na nasz aktualny SP.' }}

{{ 'Czym jest VotingPower(VP)?' }}

{{ 'Blockchain Steem pozwala nam głosować nieskończoną ilość razy, jednak z każdym 100% głosem Twój VP będzie pomniejszany o około ~ 2%. Jednak VP regeneruje się ~ 20% na dobę. Siła głosu pozwala zdecydować jak dużą część nagrody z puli chcemy przekazać autorowi, jednak jeśli posiadamy miej niż ~ 480SP zalecane jest głosować na 100%, ponieważ siła głosu jest bardzo mała.' }}

{{ 'Czym jest Bandwith?' }}

{{ 'Wszystko co robisz (dla przykładu głosujesz, dodajesz komentarz czy treść) trafia do blockchainu Steem, bandwith jest niczym innym jak limitem danych jaki możesz przesłać do Steem. Na jego wielkość wpływa Twoje SteemPower, im większe tym większy jest bandwith.' }}

{{ 'Czym jest Flaga (zgłoszenie lub ujemny głos)?' }}

{{ 'Dodając flagę obniżamy wartość danej nagrody (z treści lub komentarza) o aktualną siłę naszego głosu. Flaga również obniża reputację danego Użytkownika.' }}

{{ 'Czym jest Reputacja Użytkownika?' }}

{{ 'Każdy Użytkownik posiada reputację (wskaźnik), która jest wyrażona w liczbach obok jego nicku w nawiasach. Zdobywamy ją gdy inny Użytkownik o większej reputacji od naszej zagłosuje na naszą treść. Jest to określenie „wielkości” Użytkownika bazując na jego SteemPower. Reputację również możemy wyrazić poprzez nazewnictwo: narybek, delfin, orka lub wieloryb. Użytkownicy z niższą reputacją od naszej, nie są w stanie nam obniżyć naszej.' }}
